# ---------------------------------------------------------------------
#
# Ubuntu/Debian:
#
# The game needs the SDL library. The name of the package is: 
  libsdl1.2debian
# You can install the SDL library with the command:
# sudo apt-get install libsdl1.2debian
# or with aptitude.
#
# ---------------------------------------------------------------------
#
# Suse Linux/ Red Hat / other Linuxes:
#
# Please install the SDL with the package manager of 
# your system.
#
#
# ---------------------------------------------------------------------
#
# ATI users:
#
# The game seems not to work in fullscreen-mode with
# some ATI-cards.
#
# You can switch to window-mode with commandline-parameter:
# -w1024x768
#
